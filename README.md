# RRSSB Demo Site
Demo site for [Ridiculously Responsive Social Sharing Buttons](https://github.com/kni-labs/rrssb) which lives [here](http://rrssb.ml/).

## Setup
- make sure [node.js](http://nodejs.org) and [roots](http://roots.cx) are installed
- run `npm install`
- run `bower install`
- run `roots watch`
